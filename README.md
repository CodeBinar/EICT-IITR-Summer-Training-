# 🚀 EICT IIT Roorkee Summer Training Projects

Welcome to my repository containing the projects completed during the **Electronics & ICT Academy (EICT), IIT Roorkee Summer Training Program**.

This repository showcases the practical implementation of **Machine Learning**, **Multi-Agent AI Systems**, and **Deep Learning** through three industry-oriented projects. Each project demonstrates the application of modern AI frameworks, data analysis techniques, and intelligent automation to solve real-world problems.

The EICT IIT Roorkee Summer Training Program focused on providing hands-on experience in Artificial Intelligence, Machine Learning, Deep Learning, and Agentic AI. # 📂 Projects Included

## 1️⃣ Decision Tree Machine Learning Workflow

**Folder:** `01-Decision-Tree`

### Overview
An end-to-end Machine Learning workflow implementing Decision Tree Regression and Classification on movie datasets.

### Key Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Decision Tree Regression
- Decision Tree Classification
- Feature Importance Analysis
- Pre-Pruning
- Post-Pruning
- Cross Validation
- Hyperparameter Tuning (GridSearchCV)
- XGBoost Comparison

**Technologies Used**

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- XGBoost

---

## 2️⃣ Multi-Agent Analytics Assistant

**Folder:** `02-Multi-Agent-Analytics-Assistant`

### Overview

A production-inspired Multi-Agent AI system that simulates an analytics team using a Supervisor Agent, Data Analyst Agent, and Data Scientist Agent. The system performs dataset profiling, KPI recommendation, SQL assistance, and machine learning guidance using local Large Language Models.

### Key Features

- Multi-Agent Collaboration
- CrewAI Framework
- Hierarchical Task Delegation
- Model Context Protocol (MCP)
- Local LLM using Ollama
- Streamlit Interface
- FastAPI Backend
- SQLite Chat History
- Docker Deployment

**Technologies Used**

- Python
- CrewAI
- Ollama
- Streamlit
- FastAPI
- MCP SDK
- SQLite
- Docker

---

## 3️⃣ Resume-JD Deep Neural Network Match Scorer

**Folder:** `03-Resume-JD-Deep-Neural-Network-Match-Scorer`

### Overview

A Deep Learning-based application that evaluates the similarity between a candidate's resume and a job description using neural network techniques. The system assists recruiters by estimating candidate-job compatibility through semantic analysis.

### Key Features

- Resume Parsing
- Job Description Analysis
- Deep Neural Network Model
- Similarity Score Prediction
- Candidate Ranking
- Interactive User Interface

**Technologies Used**

- Python
- TensorFlow / Keras
- Pandas
- NumPy
- Scikit-learn
- Streamlit
