# Decision Tree Machine Learning Workflow

## 📌 Overview
This project was completed as part of the **EICT IIT Roorkee Summer Training Program**. It demonstrates an end-to-end Decision Tree Machine Learning workflow for both regression and classification tasks.

## 🎯 Objectives
- Perform data preprocessing
- Build Decision Tree Regression and Classification models
- Apply pre-pruning and post-pruning
- Perform hyperparameter tuning using GridSearchCV
- Evaluate models using cross-validation
- Compare Decision Tree with XGBoost

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost

## 📂 Dataset
- Movie Regression Dataset
- Movie Classification Dataset

## 🚀 Features
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Decision Tree Regression
- Decision Tree Classification
- Feature Importance
- Pre-Pruning
- Post-Pruning
- Cross Validation
- GridSearchCV
- XGBoost Comparison

## 📈 Results
- Built accurate Decision Tree models.
- Reduced overfitting using pruning techniques.
- Improved model performance through hyperparameter tuning.
- Compared Decision Tree performance with XGBoost.
